# Research

* https://docs.espressif.com/projects/esp-idf/en/latest/esp32/api-reference/peripherals/spi_master.html - Official docs for SPI communication for the SPI master node
* https://docs.espressif.com/projects/esp-idf/en/latest/esp32/api-reference/peripherals/spi_slave.html - Official docs for SPI communication for the SPI slave node
* https://randomnerdtutorials.com/esp32-spi-communication-arduino/ - Written in Arduino, do not use these code samples, but lots of good theory about connecting devices over SPI
* https://github.com/risinek/esp32-wifi-penetration-tool/blob/master/components/wifi_controller/sniffer.c - This is an interface for listening to WiFi frames in promiscious mode
* https://api.wigle.net/csvFormat.html - This is the formatting we need to follow to submit our findings to Wigle.net
